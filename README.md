# k-means-image-compression
This code compresses a simple parrot image using a from scratch implementation of k-means clustering.

Required libraries:

* numpy as np
* matplotlib.pyplot as plt
* cv2

IMPORTANT NOTE To run this code in your environment you must allow Google Colab to mount to your drive. Before running the code you must change your base directory to the location of the data files in the section labelled "CHANGE BASE DIRECTORY HERE TO YOUR GDRIVE FILE LOCATION" in the first code block.
